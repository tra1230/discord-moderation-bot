# Discord Moderation Bot

A comprehensive Discord bot with advanced moderation features, AI capabilities, and ML-powered detection.

## Features

- **Moderation Commands**: warn, kick, ban, mute, unmute
- **Health Checks**: ping and status commands
- **Event Listeners**: member join notifications
- **AI Integration**: Coming soon
- **ML Detection**: Coming soon
- **Dashboard & Logs**: Coming soon

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Discord bot token to `.env` file
4. Run the bot: `python main.py`

## Deployment

This bot is deployed on Railway for 24/7 uptime.
